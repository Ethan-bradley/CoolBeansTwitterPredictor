Twitter API Tweet Predictor


Ciao! This is Team Cool Bean's first time at Cal Hacks. Members include: Anthony Ebiner, Ethan Bradley, Evan Gao, and Justin Chen. We are so happy to share our wonderful project with you! Our project utilizes the Twitter API to calculate how many tweets a user sends on average per week. We wanted to estimate how many tweets a user may make in a given week. The idea stemmed from this bidding post in Predit.it. With President's Trump's tweets as an example, we calculated how many tweets he sends per day in the past and compiled this data into a large list. After averaging this data, we can then estimate how many future tweets per a week he may make.
